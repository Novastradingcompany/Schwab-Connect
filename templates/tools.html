Schwab Token Workflow
Do not rely on an in-app token refresh button. Schwab auth is a browser-based OAuth flow and the reliable path is to run connect.py locally, then update Render env TOKEN_JSON_B64.
Recommended flow:
1. Run python connect.py on your local machine.
2. Complete the Schwab login and approval flow in the browser.
3. Confirm your local token.json is updated.
4. Use Show TOKEN_JSON_B64 Value to generate the Render-safe base64 string.
5. Paste that value into Render env TOKEN_JSON_B64.
6. Redeploy the service.
